# Furniture Sprites

This folder is for PNG furniture sprites served by Vite from `/assets/furniture/...`.

Do not commit copyrighted furniture assets from Habbo, Sulake, fan archives, or other third-party sources unless you have explicit rights to use and redistribute them.

For local private prototyping, temporary reference assets may be placed here on your machine, but they should not be published in a public repository or public demo.

For portfolio, production, or public demos, use original assets created for Hotel Exchange or properly licensed assets.

Current local prototype sprites are referenced from `frontend/src/game/data/furnitureCatalog.ts` with short, stable filenames. If a source image has a non-transparent preview background, create a cleaned PNG copy instead of pointing the catalog at the raw source file.
